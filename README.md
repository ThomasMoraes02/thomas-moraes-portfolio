# Portfólio Profissional | Thomas Moraes

Landing page profissional de Thomas Vinicius de Moraes, construída com React, Vite e TypeScript.

## Rodando localmente

```bash
npm install
npm run dev
```

## Validação

```bash
npm run lint
npm run build
```

## Deploy gratuito recomendado

### Cloudflare Pages

1. Crie um repositório no GitHub para este projeto.
2. Conecte o repositório em Cloudflare Pages.
3. Use `npm run build` como comando de build.
4. Use `dist` como diretório de saída.
5. Configure a branch `main` como produção.

### Vercel

1. Importe o repositório do GitHub na Vercel.
2. Mantenha o framework como Vite.
3. Use `npm run build` como build command.
4. Use `dist` como output directory.
